# Vendored Mozilla PDF.js viewer

This directory contains the production files required by the theorem reader
from the official legacy distribution of Mozilla PDF.js release `v6.1.200`
(published 2026-06-27). Mozilla's legacy bundle supports Chrome 118 and later;
the viewer UI and feature set are the same as the modern bundle.

Source release:
<https://github.com/mozilla/pdf.js/releases/tag/v6.1.200>

The upstream Apache 2.0 license is preserved in `LICENSE`.
